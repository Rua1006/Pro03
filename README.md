# MVC Level2 Project
문화관광 + Maven MVC Project + MariaDB

### 📢 프로젝트 설명 
- 개인프로젝트
  - 문화관광 지역: 대구를 주제로 진행한 개인프로젝트
- 구현 목표
  - Controller, Model, DTO를 활용 데이터 포워딩으로 
    관광지CRUD, 공지 CRUD, 리뷰, 댓글기능 구현
    
### ⚙ 개발 환경
- 운영체제 : Windows-11
- 통합개발환경(IDE) : Eclipse
- JDK version : JDK 1.8
- 데이터베이스 : MariaDB
- 관리 툴 : GitHub

- 백엔드

- 프론트엔드<br>
